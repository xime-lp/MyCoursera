Cascading Style Sheets Basics

Power of CSS

CSS provides styling capability.

It creates an user experience that matters.

Anatomy of a CSS Rule

We have a selector followed by curly braces

Inside it we have declarations that consist of a property and a value

The browser comes with default styles for things.

Element, Class and ID Selectors

An element selector applies to thing like p, h1, div, etc.

A class selector is specified by a .classname it will target every element with that class

An id selector is specified by a #idname it will target the element with that id

Combining Selectors

We can combine element selectors with class selectors: p.classname

We can also do a child selector: article > p will target all p elements that are direct childs of an article element

Descendant selector: article p will target all p elements that are inside an article element at any level.

The child and the descendant selectors also work with classes not only with elements.

Pseudo-Class Selectors

This is for specific properties

We put the selector followed by the pseudo-class selector: p:hover

We can use selectors such as:

  • :link
  • :visited
  • :hover
  • :active
  • :nth-child()

This can be used along with anything.
