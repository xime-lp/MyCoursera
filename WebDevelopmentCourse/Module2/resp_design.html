Responsive Design

Design for all mobile devices.

Use % for sizes.

The most common column layout used nowadays is a 12 column grid responsive layout. We can also nest them.

By default, mobile browsers zoom out when viewing websites. To stop this we use a meta tag that says { meta name="viewport" content="width=device-width, initial-scale=1" }
