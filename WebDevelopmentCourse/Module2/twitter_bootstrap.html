Introduction to Twitter Bootstrap

Setup

  • Download Bootstrap file from webpage
  • Download jquery
  • Save the file on Documents
  • Duplicate the needed files and include them in your project file
  • Link them to your project
  • For CSS use bootstrap.min.css
  • To use the JS files you need to include jquery
  • Link jquery first and then bootstrap.min.js
  • ( js is linked using script )

The Bottstrap Grid System

Based on the 12-grid system. Setup:

  • Grids should be inside a "container" that is a div with class container. The class "container" has predetermined fixed responsive widths. The class "container-fluid" covers the full width of the browser and provides consistent padding around the content.
  • After the container we put a div with class "row" and this creates a horizontal group. To add another row we should use this div again. It has negative margin to counteract container padding so things line up.
  • After the row we put the columns. These should have a determined class based on the 12-grid system. Every class is defined using this template:
  • xs classes never collapse
