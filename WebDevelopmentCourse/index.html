Git process

  1. Make a new repository in github.com
  2. Go to the settings of the repository and search for GitHub Pages
  3. Change the source to master branch
  4. Open the terminal and get into the folder where you want the project to live on the computer
  5. Type 'git clone (link for cloning copied from github)' and enter
  6. Make new files or edit existing ones
  7. Type 'git add .' or 'git add [file]' to mark everything to be commited
  8. Type 'git commit -m "Add a small message here" to commit stuff
  9. Now type 'git push' to upload everything. Password and username may be required.
